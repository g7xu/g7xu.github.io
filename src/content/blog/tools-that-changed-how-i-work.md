## The Stack That Stuck

After cycling through dozens of tools during my time at UCSD and in internships, a small set proved genuinely worth keeping. This post covers what made the cut and why.

### Obsidian for Thinking Out Loud

Before Obsidian, notes lived in Google Docs — isolated, unsearchable, and forgotten. The graph view changed how I approach learning: writing a note now means _connecting_ it to what I already know.

### Cursor for Writing Code

Cursor's `Cmd+K` inline edits have eliminated the copy-paste loop between a browser and an editor. For boilerplate-heavy work (FastAPI routes, Pandas pipelines), it saves 20–30 minutes per session.

### GitHub Actions for Everything Automated

Automating deployments via Actions removed the "it works on my machine" anxiety entirely. If the workflow passes, it ships. If it fails, the error is visible to the whole team.

## What Didn't Work

- **Notion as a second brain** — Too much friction to create, too much friction to retrieve.
- **Vim** — The learning curve is real. Worth it only if your hands live on the keyboard.
